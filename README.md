# 基于Python的人脸识别智能门禁系统 python1125

## 项目概述

本项目是一个基于人脸识别技术的智能门禁管理系统，适用于实验室、公司等场所的人员进出管理。系统运用Python和Django框架，结合人脸识别算法，实现高效、便捷的管理。

## 技术栈

- Python
- Django
- MySQL
- 人脸识别技术

## 系统角色

- 系统管理员：负责用户管理、门禁设置等。
- 普通用户：进行注册、信息录入、预约实验室、考勤签到。

## 运行环境

- 系统环境：Windows
- 开发环境：Python、Django、MySQL

## 功能模块

### 用户注册与管理

- 支持用户注册、信息录入。
- 权限分配与管理。

### 人脸识别门禁

- 实时录入人脸信息。
- 自动实现门禁的开关。

### 实验室预约与考勤管理

- 提供实验室预约功能。
- 支持人员考勤签到。
- 预约记录查询与统计。
